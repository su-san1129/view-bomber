Features

Everything you need to
view any file

No more juggling apps. One viewer handles documents, data, code, images, and geospatial files.

30+ Formats

From PDF and DOCX to SQLite and GeoJSON. One app opens documents, data, images, and code files.

Folder Browser

Browse directory trees with live file watching. Changes on disk are reflected instantly.

Universal Search

Full-text search across text-based files. Find content quickly with Cmd+F and sidebar search.

Plugin Architecture

GeoJSON maps, CSV tables, syntax-highlighted code, and more. Each format gets a specialized renderer.

PDF Export

Convert Markdown to PDF with bundled pandoc and tectonic. No external tools required.

Native Performance

Built with Tauri and Rust. Lightweight binary, fast startup, and low memory usage.
