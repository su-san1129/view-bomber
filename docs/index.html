What is View Bomber?

View Bomber is a local desktop document viewer built with Tauri + React. It opens many file types in one app with fast preview, folder browsing, and search.

Multi-format Viewer Markdown, PDF, JSON/GeoJSON, CSV/TSV, DXF, DOCX, images, text, and more.
Fast Local Workflow Open folders, switch files quickly, and search inside text-based content.
Desktop Native Runs on macOS, Windows, and Linux with platform installers.
